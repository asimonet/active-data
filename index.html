Active Data now reports to you
Active Data is a free software system that tracks the life cycle of data distributed across heterogeneous software and infrastructures. It allows you to instantly watch the state of your data in real-time without digging in dozens of log files and terminals. With Active Data never lose track of your data, wherever it is!
