Get Active Data

We advise to always use the last version of Active Data, which is currently 0.2.0. The previous versions are only kept here for reference.

Build instructions

If you chose to compile the source, you need to have Apache Ant installed. Then, simply unpack the archive, go to its root and run ant dist. This will compile the jar library and javadoc, run the unit tests and place the resulting files in the dist directory.
